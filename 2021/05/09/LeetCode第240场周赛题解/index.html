人口最多的年份

给你一个二维整数数组$logs$,其中每个$logs[i] = [birthi, deathi]$表示第$i$个人的出生和死亡年份。

年份$i$的人口定义为这一年期间活着的人的数目。第$i$个人被计入年份$x$的人口需要满足：$x$在闭区间$ [birthi, deathi - 1] $内。注意，人不应当计入他们死亡当年的人口中。
返回 人口最多 且 最早 的年份。

题解

直接模拟即可

class Solution {
    int st[2100];
public:
    int maximumPopulation(vector<vector<int>>& logs) {
        for (auto it:logs) 
            for (int k = it[0]; k <= it[1] -1; k ++) {
                st[k] ++;
            }
        int ans = 0, sta = 0;
        for (int i = 1950; i <= 2050; i ++) {
            if (st[i] > sta) {
                sta = st[i];
                ans = i;
            }
        }
        return ans;
    }
};

下标对中的最大距离

给你两个 非递增 的整数数组 $nums1$​​​​​​ 和 $nums2$​​​​​​ ，数组下标均 从 $0 $开始 计数。

下标对$ (i, j)$ 中$ 0 <= i < nums1.length$ 且 $0 <= j < nums2.length $。如果该下标对同时满足 $i <= j$ 且 $nums1[i] <= nums2[j] $，则称之为 有效 下标对，该下标对的 距离 为$ j - i​$​ 。​​

返回所有 有效 下标对 $(i, j)$ 中的 最大距离 。如果不存在有效下标对，返回 0 。

一个数组 $arr$ ，如果每个 $1 <= i < arr.length$ 均有 $arr[i-1] >= arr[i]$ 成立，那么该数组是一个 非递增 数组

题解

注意到它给我们的数组是具有单调性的。那么这个时候我们可以考虑一下双指针扫描一下，因为随着一个数组内指针的变化，另外一个数组内指针必定也是同方向偏移。


class Solution {
public:
    int maxDistance(vector<int>& nums1, vector<int>& nums2) {
        int res = 0;
        for (int j = nums2.size() - 1, i = nums1.size() - 1;j >= 0; j --) {
            while (i && nums1[i - 1] <= nums2[j]) i --;
            if (nums1[i] <= nums2[j]) res = max (res, j - i);
        }
        return res;
    }
};

子数组最小乘积的最大值

一个数组的 最小乘积 定义为这个数组中 最小值 乘以 数组的 和 。

比方说，数组 $[3,2,5] $（最小值是 2）的最小乘积为 $2 * (3+2+5) = 2 * 10 = 20 $。
给你一个正整数数组$ nums$ ，请你返回 $nums $任意 非空子数组 的最小乘积 的 最大值 。由于答案可能很大，请你返回答案对  $1e9 + 7$ 取余 的结果。

请注意，最小乘积的最大值考虑的是取余操作 之前 的结果。题目保证最小乘积的最大值在 不取余 的情况下可以用 $64$ 位有符号整数 保存。

子数组 定义为一个数组的 连续 部分

题解

对于每一个位置来说，左右两边大于它的位置都是确定的，那么我们答案一定是在某个值对应的左右区间乘上其本身的。换句话说我们枚举以每个本身为最小值的区间，然后取最小值，这个枚举通过单调栈来实现。

typedef long long ll;
const int maxn = 1e5 + 10;
const int mod = 1e9 + 7;
ll pre_sum[maxn];
int l[maxn], r[maxn], h[maxn];
vector <int> v;

class Solution {
public:
    int maxSumMinProduct(vector<int>& nums) {
        int n = nums.size();
        for (int i = 1; i <= n; i ++) {
            h[i] = nums[i - 1];
            pre_sum[i] = pre_sum[i - 1] + h[i];
        }
        h[0] = h[n + 1] = 0;
        v.push_back (h[0]);
        for (int i = 1; i <= n; i ++) {
            while (h[i] <= h[v.back()]) v.pop_back();
            l[i] = v.back();
            v.push_back (i);
        }
        v.clear();
        v.push_back(n + 1);
        for (int i = n; i >= 1; i --) {
            while (h[i] <= h[v.back()]) v.pop_back();
            r[i] = v.back();
            v.push_back (i);
        }
        ll res = 0;
        for (int i = 1; i <= n; i ++) {
            res = max (res, h[i] * (pre_sum[r[i] - 1]-pre_sum[l[i]]));
        }
        return res % mod;
    }
};

有向图中最大颜色值

给你一个 有向图 ，它含有 $n$ 个节点和 $m$ 条边。节点编号从 $0$ 到 $n - 1 $。
给你一个字符串 $colors$ ，其中 $colors[i]$ 是小写英文字母，表示图中第 i 个节点的 颜色 （下标从 0 开始）。同时给你一个二维数组 $edges $，其中 $edges[j] = [aj, bj] $表示从节点 $aj $到节点$ bj$ 有一条 有向边 。

图中一条有效 路径 是一个点序列 $x1 -> x2 -> x3 -> … -> xk$，对于所有 $1 <= i < k $，从 $xi$ 到 $xi+1$ 在图中有一条有向边。路径的 颜色值 是路径中 出现次数最多 颜色的节点数目。

请你返回给定图中有效路径里面的 最大颜色值 。如果图中含有环，请返回 -1 。

题解

首先我们考虑一下，如何搞这个最大颜色路径，可以想到的是dp转移。那么他还有一个要求就是在有环时返回$-1$，那么我们可以考虑拓扑排序，实际上我们dp的时候也是要在拓扑序的基础上操作的，因为在某点时要保证前方可达。$dp[i][c]$到表示第i个点的路径中颜色为c的最大颜色路径，状态转移为$dp[v][c]=max (dp[v][c],dp[u][c] + (1/0))$

const int maxn = 1e5 + 10;
vector<int> G[maxn];
int n, m;
int dp[maxn][30];
int in_d[maxn];

class Solution {
public:
    int largestPathValue(string colors, vector<vector<int>>& edges) {
        n = colors.length();
        for (int i = 0; i <  n; i ++) 
            for (int c = 0; c < 26; c ++) dp[i][c] = 0;
        for (int i = 0; i < n; i ++) in_d[i] = 0, G[i].clear();
        queue<int> q;
        while (q.size()) q.pop();
        for (auto it:edges) {
            int u = it[0], v = it[1];
            in_d[v] ++;
            G[u].push_back (v);
        }
        for (int i = 0;i < n; i++) {
            if (in_d[i] == 0) {
                q.push (i);
                dp[i][colors[i] - 'a'] += 1;
            }
        }
        int st = 0;
        while (q.size()) {
            int now = q.front();
            q.pop ();
            st ++;
            for (auto it:G[now]) {
                in_d[it] --;
                for (int c = 0; c < 26; c++) dp[it][c] = max(dp[it][c], dp[now][c]);
                if (in_d[it] == 0) {
                    q.push(it);
                    dp[it][colors[it] - 'a'] += 1;
                }
            }
        }
        if (st < n) return -1;
        int ans = 0;
        for (int i = 0; i < n; i ++) 
            for (int c = 0; c < 26; c++) {
                ans = max(ans, dp[i][c]);
            }
        return ans;
    }
};
